爆破

用户名一般都是root，有一个工具phpmyadmin多批量线程批量破解工具，进行爆破。

进入phpmyadmin

  • 数据库服务器信息
  • 网站服务器信息

写shell到服务器 通用漏洞

  • 确定服务器真实路径 phpinfo.php,搜索引擎,fofa,错误信息
  • 通过探针 获取 IP，php开启了那些函数(include……)

phpmyadmin4.8.1漏洞CVE-2018-12613

  • session 包含在tmp文件夹中
  • cookie中phpmyadmin会有session的名字
    sess_sessionname
  • 你需要知道session文件目录

phpmyadmin index页面 4.8.1 版本 文件包含漏洞

  • frm文件是在每个数据表被建立的时候自动简历
    select @@datadir
  • frm文件会记录你在这个表里面的操作，也就是说你在任意一个属性写入一句话，等会文件被包含就能够被执行
  • 最重要的是根据源码，对其进行审计，构造payload
  • 这个漏洞中还需要有2次url解码
